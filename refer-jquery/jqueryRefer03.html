01. 속성 메서드 : addClass() / removeClass()

클릭하면 모든 이미지에 클래스(active)를 추가합니다. 클릭하면 모든 이미지에 클래스(active)를 삭제합니다. 
{
    $(".btn1-1").click(function () {
        $(".ex1 div").addClass("active");
    });
    
    $(".btn1-2").click(function () {
        $(".ex1 div").removeClass("active");
    });
}

02. 속성 메서드 : toggleClass()

이미지를 클릭하면 클래스(active)를 추가/삭제합니다. 
{
    //모든 이미지 선택
    $(".ex2 div").click(function () {
        $(".ex2 div").toggleClass("active");
    });

    //선택한 이미지 선택
    $(".ex2 div").click(function () {
        $(this).toggleClass("active");
    });
}

03. hasClass()

리셋 클릭하면 active 클래스를 찾은 후에 클래스를 삭제합니다. 
{
    $(".btn3-0").click(function (){
        $(".ex3 > div:nth-child(1)").addClass("active");
        $(".ex3 > div:nth-child(4)").addClass("active");
    });
    
    $(".btn3-1").click(function () {
        if ($(".ex3 > div").hasClass("active")) {
            $(".ex3 > div.active").removeClass("active");
        }
    });
}

04. width, heigth

여기는 텍스트 영역입니다.

클릭하면 원의 가로(width) 값을 구합니다. 클릭하면 원의 가로(innerWidth) 값을 구합니다. 클릭하면 원의 가로(outerWidth) 값을 구합니다. 클릭하면 원의 세로(height) 값을 구합니다. 클릭하면 원의 세로(innerHeight) 값을 구합니다. 클릭하면 원의 세로(outerHeight) 값을 구합니다.
클릭하면 샘플 박스의 가로 값을 구합니다. 클릭하면 문서의 가로 값을 구합니다. 클릭하면 브라우저의 가로 값을 구합니다. 
{
    function dimension(name, size) {
        $(".text4").text("이 원의 " + name + "값은 " + size + "px 입니다.");
    }

    $(".btn4-1").click(function () {
        dimension("가로(width)", $(".circle4").width());
    });
    $(".btn4-2").click(function () {
        dimension("가로(innerWidth)", $(".circle4").innerWidth());
    });
    $(".btn4-3").click(function () {
        dimension("가로(outerWidth)", $(".circle4").outerWidth());
    });
    $(".btn4-4").click(function () {
        dimension("세로(height)", $(".circle4").height());
    });
    $(".btn4-5").click(function () {
        dimension("세로(outerHeight)", $(".circle4").innerHeight());
    });
    $(".btn4-6").click(function () {
        dimension("세로(outerHeight)", $(".circle4").outerHeight());
    });
    
    //샘플 박스 사이즈 구하기
    $(".btn4-7").click(function () {
        $(".text4").text("이 샘플 박스의 가로 값은 " + $("#sample04").width() + "px 입니다.");
    });

    //문서 사이즈 구하기
    $(".btn4-8").click(function () {
        $(".text4").text("이 문서의 가로 값은 " + $(document).width() + "px 입니다.");
    });

    //브라우저 사이즈 구하기
    $(".btn4-9").click(function () {
        $(".text4").text("이 브라우저의 가로 값은 " + $(window).width() + "px 입니다.");
    });
}

05. width()

리셋 클릭하면 원의 가로 값을 300px로 변경합니다. 클릭하면 원의 세로 값을 300px로 변경합니다. 클릭하면 원의 가로/세로 값을 200px로 변경합니다. 
{
    $(".btn5-0").click(function () {
        $(".circle5").width("70").height("70");
    });
    $(".btn5-1").click(function () {
        $(".circle5").width("300");
    });
    $(".btn5-2").click(function () {
        $(".circle5").height("300");
    });
    $(".btn5-3").click(function () {
        $(".circle5").width("200").height("200");
    });
}

06. offset(), position(),

여기는 텍스트 영역입니다.

클릭하면 원의 x축 위치 값(offset)을 구합니다. 클릭하면 원의 y축 위치 값(offset)을 구합니다. 클릭하면 원의 x축 위치 값(position)을 구합니다. 클릭하면 원의 y축 위치 값(position)을 구합니다. 
{
    $(".btn6-1").click(function () {
        $(".text6").text("이 원의 x축 값은 " + $(".circle6").offset().left + "px 입니다.");
    });
    $(".btn6-2").click(function () {
        $(".text6").text("이 원의 y축 값은 " + $(".circle6").offset().top + "px 입니다.");
    });
    $(".btn6-3").click(function () {
        $(".text6").text("이 원의 x축 값은 " + $(".circle6").position().left + "px 입니다.");
    });
    $(".btn6-4").click(function () {
        $(".text6").text("이 원의 y축 값은 " + $(".circle6").position().top + "px 입니다.");
    });
}
