릭 쿡(Rick Cook)

“오늘날의 프로그래밍은 크고 아름다운 멍청이가 써도 고장나지 않는 프로그램을 구축하려 애쓰는 소프트웨어 공학자와 그걸 망칠 수 있을 정도로 더 큰 멍청이를 만들려는 우주의 경쟁이다. 지금까지는 우주가 이기고 있다.”

에릭 S 레이먼드(Eric S. Raymond)

“붓과 물감을 공부한다고 해서 누구나 전문 화가가 될 수 있는것이 아니듯, 컴퓨터 과학 교육이 아무나 전문 프로그래머로 만들지 못한다.”

빌 클린턴(Bill Clinton)

“우리 컴퓨터 프로그램의 안타까운 현 상태를 논한다면, 소프트웨어 개발은 여전히 흑마술이 분명하며, 아직 공학이라고 부를 수 없다.”

앨런 펄리스(Alan J. Perlis)

“누군가가 '원하는 것을 얘기 하기만 하면 이루어지는 프로그래밍 언어를 원한다'고 말하면 그에게 막대 사탕을 주도록 하라.”

소프트웨어 공학에서의 모셔의 법칙(Mosher’s Law)

“올바로 동작하지 않더라도 걱정말아라. 모든 것이 그랬다면, 넌 직업을 잃었을테니까.”

const tabBtn = document.querySelectorAll(".tab-btn Ul li");
const tabCont = document.querySelectorAll(".tab-cont > div");

tabBtn.forEach((element, index) => {
	element.addEventListener("click", () => {
		tabBtn.forEach(el => {
			el.classList.remove("active");
		});
		element.classList.add("active");

		tabCont.forEach(el => {
			el.style.display = "none";
		});

		tabCont[index].style.display = "block";
	});
});
